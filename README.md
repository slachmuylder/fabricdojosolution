## How to get started:
1. Click the green "Use this template" button at the top right of this repository
2. Make sure you have the correct Owner account selected 
3. Name your repository and choose public/private
4. Click "Create repository"
5. Clone YOUR new repository to your local machine:
```
   git clone https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
   cd YOUR-REPO-NAME
```
